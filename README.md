## Overview

Spin up an ESP32 HTTP server to expose endpoints for device control and status. This example provides `/` to read LED state and `/toggle` to switch it.

## Code Explained

- Initializes GPIO2 as output for the onboard LED.
- Starts the ESP-IDF HTTP server and registers two routes.
- Responds with simple text payloads suitable for testing.

## How To Use

- Flash to ESP32 (ESP-IDF). Ensure Wi-Fi provisioning separately.
- Open a browser to `http://<device_ip>/` and `http://<device_ip>/toggle`.
